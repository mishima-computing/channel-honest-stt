# 17. レベル2（背景雑音テスト）：9分類体制でのSNR耐久曲線の再測定

レベル1における「井戸の整理」を経て確定した強固な9分類体制（子音4＋母音5）の真価を問うため、再度レベル2（Pink Noise重畳）のSNR耐久テストを実施した。

## 17.1 測定条件
- **分類体制**: 全9分類（弾き音、有声破擦、有声摩擦を有声破裂へ統合）
- **雑音種**: Pink Noise
- **SNR条件**: Clean (∞), 20dB, 10dB, 5dB, 0dB
- **使用窓**:
  - 子音: 後続母音アンカー `[-80ms, 0ms]`（※過渡音は `[-20ms, +40ms]` バースト基準を継続検討中だが、今回のベースライン測定では統一して検証）
  - 母音: 母音アンカー `[0ms, +80ms]`

## 17.2 測定結果：全体精度の向上
脆い井戸（弾き音など）を強固な井戸（有声破裂）へマージした結果、全体の正答率とSNR耐性に明確な向上が見られた。

- **Clean (雑音なし)**: 約72% → **72.85%** へ微増
- **SNR 0dB (最大雑音)**: 55.43% → **57.86%** へ向上 (+2.4ポイント)

「線形分離できない井戸を無理に分けず、統合して巨大な井戸を作る」という本規格の基本戦略が、雑音下でのモデル全体の崩壊を防ぐ効果（堅牢性の向上）に直結することが実証された。

## 17.3 9分類SNR曲線の考察
![SNR Robustness Curve (9-class)](../assets/snr_robustness_curve_9class.png)

- **有声破裂（巨大化した井戸）の挙動**:
  弾き音、有声破擦、有声摩擦を吸収して巨大化した「有声破裂」井戸が、雑音下においてどのような耐久性（曲線の傾き）を示すかが最大の関心事であった。
  （※曲線の詳細な読み解きは次節の議論にて行う）

- **母音と他の子音群**:
  /u/ を含む母音群や、無声破裂・鼻音といった他の強固な井戸についても、全体の編成が変わったことで混同先への散り方が変化している可能性がある。

## 17.4 今後の留保事項
今回のレベル2再測定は、前回の留保を引き継いでいる。
1. **雑音種の拡張**: 現在はPink Noise（定常・低域寄り）のみの検証であり、将来的にはMUSAN等の非定常雑音での検証が必要である。
2. **Matched学習の限界**: 雑音下での学習データを含む「環境最適化」状態での性能上限を見ている点に留意する。
