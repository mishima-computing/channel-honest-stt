# 結論の位置づけ修正と混同行列フェーズへの移行

## 1. 「分離可能」という結論の再定義（最良条件での結果）
「03_formant_synthesis_debug_and_validation」にて、全7母音（u-o等の境界含む）の谷深さが30dB以上残存し、「分離可能（畳まない）」という結論を出した。
しかし、この結論は**「最良条件（素の合成音、雑音なし、残響なし、tanhドライブ控えめ）」における局所的な結果**に過ぎない。

システム設計の背骨である「最悪ケースに合わせる」という原則に立ち返れば、実運用環境（フェーズ2の背景雑音・残響、フェーズ3のノイズ抑圧やAGC等の送話側信号処理）が加わった際、相互変調歪みやノイズによって谷が埋まり、分離不可能になるリスクは依然として残っている。
また、平均パワーによる谷深さ測定は、F0倍音による局所的なスパー（相互変調ノイズの峰）の上昇を過小評価している可能性がある。

**【修正された結論】**
「素のチャンネル（フェーズ1）においては、7井戸すべてが分離可能であり、現時点でのマージは不要である。しかし、これを最終確定とせず、フェーズ2（雑音等）およびフェーズ3（信号処理）の劣化を段階的に加えた『混同行列』によって再評価し、最終的なマージ判定（畳む/残す）を下す。」

## 2. 混同行列フェーズへの移行計画
手作業による谷深さ測定（簡易検証）を終え、本格的な「劣化込みで井戸が分離可能か」を判定するツールである混同行列の構築へ進む。
7井戸は固定された確定値ではなく、「暫定の出発点」として扱う。

### 段階的混同行列（劣化フェーズの適用）
1. **フェーズ1（ベースライン）**: 
   素のチャンネル（8kHzダウンサンプル＋μ-law＋500HPF＋tanh）を通した状態での混同行列を作成。まずは母音7井戸のみでパイプラインを確立する。
2. **フェーズ2（雑音・残響）**: 
   背景雑音や残響を印加し、混同行列がどのように崩れるか（特にu-o等の境界が混ざり始めるか）を観察する。
3. **フェーズ3（送話側処理）**: 
   ノイズリダクションやAGC等の非線形処理を近似追加し、最終的な分離限界を見極める。

### マージ判定の最終プロセス
フェーズ1〜3を通した混同行列の「誤判定率（Confusion Rate）」を根拠とし、最終的なクラス解像度（21井戸からどこまで畳むか）をデータ主導で確定する。子音（摩擦・破裂など）の合成手法についても、母音パイプライン確立後に順次設計・統合していく。
