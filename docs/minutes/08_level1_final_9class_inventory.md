# 16. レベル1（クリーン検証）総括：21音素から9分類へのマージ履歴と根拠の棚卸し

レベル2（背景雑音テスト）への本格移行を前に、これまでのレベル1（背景雑音なし・電話帯域劣化のみ）における検証の軌跡と、多数のマージ（統合）判断を本議事録にて総括する。
この「9分類」は、実音声（JVS）と線形分類器による数々のアーティファクト排除・物理的限界の証明を経て鍛え上げられた、本規格における「最も強固な井戸（音素カテゴリ）の定義」である。

## 16.1 本検証の目的と「マージ」の哲学
本規格の音声認識は、あえて「弱い線形分類器」を用いることで、電話帯域（8kHz、μ-law、500Hz HPF）という過酷な物理的制約のもとで**「音響的に絶対に分離できない境界線（本質的近さ）」**を可視化してきた。
ディープラーニング等の「強い分類器」を用いれば無理に分離できるかもしれない微小な差異であっても、クリーン環境（レベル1）の時点で線形分離できないほど浅い井戸は、雑音環境（レベル2）では必ず最初に崩壊する。
したがって、**「線形分類できない音は、物理的限界としてマージ（統合）し、一つの巨大で強固な井戸を作る。足りない解像度は上位レイヤー（大脳・文脈）で補完する」**というのが本規格の哲学である。

## 16.2 21音素から9分類への統合履歴と物理的根拠

当初設定された日本語の基本的な21音素（母音5 ＋ 子音16）は、以下の検証を経て9つの強固な分類へと整理された。

### ① 母音（5分類）の維持と /u/ の処遇
- **結果**: `/a/, /i/, /u/, /e/, /o/` の5分類を維持。
- **/u/ の脆さと維持の理由**:
  - /u/ は正答率47%と低く、全方位に誤分類が散る「本質的な脆さ」を見せた。これは日本語の /u/ のエネルギーが弱く、さらに 500Hz HPF によってF1（第1フォルマント）が削り取られるためである（物理的限界）。
  - しかし、/u/ は日本語の母音として必須であり、子音等へ統合することは不可能であるため、独立した井戸として**維持**する。
  - この脆さは、F1を信頼せずF2で弁別すること、および上位レイヤー（大脳＋確認フロー）による自動発火・補完によってシステム全体で吸収する（議事録6.4の既定方針）。

### ② 無声破擦（ts, ch） → 無声破裂（p, t, k）へのマージ
- **物理的根拠**: 
  - 過渡音（バースト基準）や80ms窓など、摩擦部を完全に包含する窓で抽出しても、無声破擦の分離度は壊滅的であり、大半が無声破裂へ流れた。
  - 独立した長い「摩擦音（s, sh等）」は中域に情報があり電話帯域を生き残るが、破擦音の短い摩擦部は高域（4kHz以上）に依存しているため、電話帯域（4kHz上限）では情報量を失い、ただの破裂音と区別がつかなくなることが証明された。

### ③ 有声破擦・摩擦（z, j） → 有声破裂（b, d, g）へのマージ
- **物理的根拠**:
  - JVSのラベル上統合されている `z, j` をコンテキスト（直前が母音か無音か）で代理分離し、単独の「有声摩擦 [z]」のみを取り出して診断した。
  - 破裂的な立ち上がりが弱いはずの有声摩擦 [z] においてすら、無声摩擦のような生存を見せず、有声破裂へ大量に流入（約55%）した。
  - これは「電話帯域では有声摩擦の識別情報が失われ、有声破裂に吸い込まれる」という物理的限界の強い証拠であり、破擦・摩擦をまとめて有声破裂へマージすることが妥当と判定された。

### ④ 弾き音（r, ry） → 有声破裂（b, d, g）へのマージ
- **物理的根拠**:
  - 当初、弾き音の分離度は42%と極端に低く、誤分類の40%が先行母音へ流れていた。
  - これを「先行母音の吸い込みアーティファクト」と断定し、弾き音のラベル開始位置をアンカーとした厳密な専用窓（`[-10ms, +30ms]`）でアーティファクトを完全に排除して再測定した。
  - その結果、先行母音への散りは消滅したが、正答率は44.4%に留まり、残りの誤分類の多く（16.9%）が有声破裂へ一極集中した（なお、/u/への流入も21.3%残存したが、弾き音は子音であるため母音への帰属は否定される）。
  - これは、弾き音 [ɾ] と有声破裂 [d] が「同じ歯茎での閉鎖と開放」という調音的な本質的近さを持ち、電話帯域で低域（閉鎖の手がかり）が削られると線形分離が困難になるという物理的限界を証明している。

## 16.3 最終確定した「9分類システム」

以上のマージを経て、本規格のベースラインとなる分類体系は以下の「母音5 ＋ 子音様式4 ＝ 計9分類」として確定した。

**【母音（5分類）】**
1. `/a/`
2. `/i/`
3. `/u/` （※脆いが上位レイヤーで補完）
4. `/e/`
5. `/o/`

**【子音様式（4分類）】**
6. **無声破裂・破擦** (`p, t, k, ts, ch`)
7. **有声破裂・破擦・摩擦・弾き** (`b, d, g, z, j, r, ry`)
8. **無声摩擦** (`s, sh, h, f`)
9. **鼻音** (`m, n, N`)

## 16.4 留保事項と次なる検証（レベル2）への引き継ぎ
本9分類体制は、あくまで以下の条件下で導き出されたものであることを明記し、今後の検証へ留保として引き継ぐ。
1. **線形・レベル1の結論であること**: 雑音のない帯域制限のみの環境下において、線形分類器を用いて導き出された境界である。
2. **レベル2でのマージ効果の立証**: 脆弱な井戸（破擦、摩擦、弾き）を強固な井戸（破裂）へ統合したこの9分類体制が、レベル2（背景雑音重畳）において実際に「ノイズ下でも崩壊しにくい（SNR耐性が高い）」という恩恵をもたらすか（特に巨大化した有声破裂井戸の振る舞い）を、次フェーズで実証する必要がある。
3. **雑音種の拡張**: レベル2のテストは現行のPink Noise（定常雑音）に加え、将来的にはMUSANコーパス等を用いた非定常雑音での検証が必須となる。
